# SalesPilot Web

React 19 + Vite + TypeScript + Tailwind CSS single-page application (SPA).

## Quick start

```bash
npm install
npm run dev
```

Opens http://localhost:5173. The dev server proxies `/api/*` requests to http://localhost:8080 (the Go API).

See the full documentation at [../../docs/README.md](../../docs/README.md).
